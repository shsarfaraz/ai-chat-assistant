# 🤖 AI Chat Assistant

An AI-powered chatbot built with Python, Streamlit, and the Groq API. This application allows users to chat with an AI assistant through a clean and interactive web interface.

## 🚀 Features

- AI-powered chatbot
- Fast responses using Groq
- Simple and responsive Streamlit UI
- Environment variable support (.env)
- Error handling
- Clean and modular Python code

## 🛠️ Technologies Used

- Python
- Streamlit
- Groq API
- OpenAI Python SDK
- python-dotenv

ai-chat-assistant/

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/ai-chat-assistant.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key_here
```

Run the application:

```bash
streamlit run app.py
```
